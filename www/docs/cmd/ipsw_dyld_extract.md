# ipsw dyld extract

Extract dyld_shared_cache from DMG in IPSW

```
ipsw dyld extract <IPSW> <DEST> [flags]
```

## Options

```
  -a, --dyld-arch stringArray   dyld_shared_cache architecture to extract
  -h, --help                    help for extract
```
