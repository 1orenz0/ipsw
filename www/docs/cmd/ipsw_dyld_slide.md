# ipsw dyld slide

Dump slide info

```
ipsw dyld slide <dyld_shared_cache> [flags]
```

## Options

```
  -a, --auth           Print only slide info for mappings with auth flags
  -c, --cache string   path to addr to sym cache file
  -h, --help           help for slide
      --json           Output as JSON
```
