# ipsw dyld a2o

Convert dyld_shared_cache address to offset

```
ipsw dyld a2o <dyld_shared_cache> <vaddr> [flags]
```

## Options

```
  -d, --dec    Return address in decimal
  -h, --help   help for a2o
  -x, --hex    Return address in hexadecimal
```
