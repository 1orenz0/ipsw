# ipsw dyld o2a

Convert dyld_shared_cache offset to address

```
ipsw dyld o2a <dyld_shared_cache> <offset> [flags]
```

## Options

```
  -d, --dec    Return address in decimal
  -h, --help   help for o2a
  -x, --hex    Return address in hexadecimal
```

```

```
