# ipsw kernel ctfdump

Dump CTF info

```
ipsw kernel ctfdump [flags]
```

## Options

```
  -a, --arch string   Which architecture to use for fat/universal MachO
  -h, --help          help for ctfdump
  -j, --json          Output as JSON
      --pretty        Pretty print JSON
```
